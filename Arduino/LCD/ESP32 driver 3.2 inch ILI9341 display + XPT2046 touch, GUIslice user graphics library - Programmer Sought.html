Important: If the touch screen needs to use XPT2046_TouchScreen driver independently, the line #define USE_HSPI_PORT must be uncommented, and it takes 2 days to find this line

Two, configure XPT2046

arduino IDE download: XPT2046_TouchScreen library.
After I touched it for a long time, the XPT2046 library was downloaded, but there was no response. Later, I downloaded the adafruit touchscreen driver and found that it does not support ESP32. Someone abroad has rewritten the driver, but the pins need to be reconnected.
The reason why XPT2046 did not respond before was that the configuration only had two legs:

#define TOUCH_CS_PIN       14
#define TOUCH_IRQ_PIN      27

  • 1
  • 2

But there are 5 T_IRQ, T_DO, T_DIN, T_CS, T_CLK pins touched on the back of the screen. After checking a lot of information, I realized that this driver uses ESP32 hardware SPI without pin mapping.
The wiring is as follows

Touch screen pin Touch screen pin description ESP32 pin ESP32 pin description
T_IRQ Touch screen interrupt information, low level when touch is detected 27 Customizable pins
T_DO Touch SPI bus output, no custom pin 19（MISO） VSPI pin MISO
T_DIN Touch SPI bus input, no custom pin 23（MOSI） VSPI pin MOSI
T_CS Touch chip select signal, low level enable, customizable pin 14 Customizable pins
T_CLK Touch the SPI bus clock signal, no custom pin 18（SCLK） SCLK

Touch screen test code

/* Map XPT2046 input to ILI9341 320x240 raster */
#include "SPI.h"
#include <XPT2046_Touchscreen.h> /* https://github.com/PaulStoffregen/XPT2046_Touchscreen */

#define TOUCH_CS_PIN       14
#define TOUCH_IRQ_PIN      27

XPT2046_Touchscreen touch( TOUCH_CS_PIN, TOUCH_IRQ_PIN );

void setup() {
  Serial.begin( 115200 );
  touch.begin();
  Serial.println( "Touch screen ready." );
}

TS_Point rawLocation;

void loop() {
    while ( touch.touched() )
    {
      rawLocation = touch.getPoint();
        Serial.print("x = ");
        Serial.print(rawLocation.x);
        Serial.print(", y = ");
        Serial.print(rawLocation.y);
        Serial.print(", z = ");
        Serial.println(rawLocation.z);   
    }
}
